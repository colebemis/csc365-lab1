# csc365-lab1
Python3 "database" for student information.

## Getting Started
Simply open your favorite terminal and run the following commands: 
```
$ cd src
$ python3 schoolsearch.py
```

## Supported Queries
* `S[tudent]: <lastname> [B[us]]`
* `T[eacher]: <lastname>`
* `B[us]: <number>`
* `G[rade]: <number> [H[igh]|L[ow]]`
* `A[verage]: <number>`
* `I[nfo]`
* `Q[uit]`
